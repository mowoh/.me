Unit 12: Capstone Project

Apply the principles learnt in previous units to design a modular, safe and secure solution targeting cybersecurity:

  • Covers secure coding, modular software architecture, OOP, design patterns, test driven development and UI with WPF

    • Objectives

  • Design and implement a complex software system using advanced object-oriented design principles, design patterns, and secure coding practices.
  • Apply knowledge to solve real-world problems in cybersecurity.

    • Task

    Design of the Secure Banking App

    Solution Steps

  • provide a simple banking UI using .Net WPF,
  • support multiple account types such as current accounts, savings accounts, etc.
  • persist account data with SQLlite
  • support secure transactions and storage via the use of encryption
  • encapsulate account behavior behind interfaces
  • promote mocking and testability of the Accounts classes via the use of sufficient interfaces
  • allow future extensibility through factories and abstractions
  • employment of the test-driven development approach
  • ensure thread safety using locks
